The `dice` command allows users to roll a six-sided dice and get a random number within your Discord server using Juniper Nexus Bot.

## Command Usage

To roll the dice, simply use the following command:

```
/dice
```

## Permissions

-   **Required Permissions**: None (available to all users).

## Command Behavior

-   Upon executing the command, the bot rolls a six-sided dice and generates a random number between 1 and 6.
-   The command responds with an embedded message displaying the result of the dice roll.

## Example

To roll the dice:

```
/dice
```

## Notes

-   This command provides a fun and interactive way for users to generate random numbers or play games within your Discord server.
-   It does not require any special permissions to use.
