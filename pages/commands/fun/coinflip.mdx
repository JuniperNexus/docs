The `coinflip` command allows users to flip a virtual coin to get a random outcome (heads or tails) within your Discord server using Juniper Nexus Bot.

## Command Usage

To flip a coin, simply use the following command:

```
/coinflip
```

## Permissions

-   **Required Permissions**: None (available to all users).

## Command Behavior

-   Upon executing the command, the bot generates a random outcome.
-   If the outcome is less than 0.5, the bot announces "heads". Otherwise, it announces "tails".
-   The command responds with an embedded message displaying the result of the coin flip.

## Example

To flip a coin:

```
/coinflip
```

## Notes

-   This command provides a fun and interactive way for users to make simple decisions or games within your Discord server.
-   It does not require any special permissions to use.
