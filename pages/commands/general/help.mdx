The `help` command provides assistance with commands available in Juniper Nexus Bot.

## Command Usage

To get help with commands, use the following command:

```
/help [command]
```

-   `command` (optional): Specify the command you want help with. If not provided, it lists all available commands.

## Permissions

-   **Required Permissions**: None (available to all users).

## Command Behavior

-   If no `command` is specified, the bot lists all available commands along with their descriptions.
-   If a specific `command` is specified, the bot provides detailed help including the command's description and options.

## Examples

To list all available commands:

```
/help
```

To get help for a specific command, for example, the `avatar` command:

```
/help avatar
```

## Notes

-   This command is essential for users to understand the available functionality and usage of commands within Juniper Nexus Bot.
