The `ping` command allows you to check the bot's latency and response time.

## Command Usage

To check the bot's latency and response time, use the following command:

```
/ping
```

## Permissions

- **Required Permissions**: None (available to all users).

## Command Behavior

- When executed, the bot calculates and displays:
  - **API Latency**: Latency between the bot and Discord's API.
  - **Message Latency**: Latency between when the command was received and when the bot responded.

## Example

To check the bot's latency and response time:

```
/ping
```

## Notes

- This command is useful for checking the bot's performance and latency in Discord.

---

Use the `ping` command to check the bot's latency and response time in your Discord server.
